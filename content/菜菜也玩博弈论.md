##  巴什博奕 

**问题描述**  
有一堆石子共 \(n\) 个，两人轮流取，每次至少取 \(1\) 个，最多取 \(m\) 个，取走最后一个者获胜。

**结论**  
若 n mod (m+1) = 0，则后手必胜；否则先手必胜。  
*策略：* 先手第一次取走 n mod (m+1)个，之后每次与对手凑成 m+1个。

```cpp
#include <iostream>
using namespace std;

int main() {
    int T, n, m;
    cin >> T;
    while (T--) {
        cin >> n >> m;
        if (n % (m + 1) == 0)
            cout << "second\n";
        else
            cout << "first\n";
    }
    return 0;
}
```

## 威佐夫博弈

**问题描述**  
有两堆石子，数量分别为 a,b（a≤b）。两人轮流取，可以在一堆中取任意多个，或者在两堆中取相同多个，取走最后一颗者获胜

**结论**  
令 k = b - a，若 a =（sqrt(5)+1）* k/2 ,则后手必胜（奇异局势）；否则先手必胜。  
其中黄金分割数（sqrt(5)+1）* k/2=1.618

```cpp
#include <bits/stdc++.h>  
using namespace std;  
   
using f80 = long double;  
   
const f80 au = (1.0L + sqrtl(5.0L)) / 2.0L;  
  
void solve() {  
    i64 a, b;  
    while (cin >> a >> b) {  
        if (a > b) swap(a, b);  
        i64 k = b - a;  
        i64 tmp = (i64) (k * au);  
        if (tmp == a) cout << "0\n";  
        else cout << "1\n";  
    }}  
  
  
int main() {  
    ios::sync_with_stdio(false), cin.tie(nullptr);  
  
    // init();  
  
    int T = 1;  
    //cin >> T;  
    while (T--) solve();  
  
    return 0;  
}
```

## 3. 尼姆博弈 (Nim Game)

**问题描述**  
有n 堆石子，每堆数量为 (a1, a2... an)。两人轮流取，每次可以从任意一堆中取任意多个（至少 1 个，至多全取），取走最后一颗者获胜。

**结论**  
将所有石子数异或：\(res = a1 ^a2 ^ an)  
若 \(res = 0\)，则后手必胜；否则先手必胜。  
*策略：* 先手应取走一些石子，使剩余局面异或为 0。

```cpp
#include <iostream>
using namespace std;

int a[105];

int main() {
    int n;
    while (cin >> n && n) {
        int res = 0;
        for (int i = 0; i < n; ++i) {
            cin >> a[i];
            res ^= a[i];
        }
        if (res == 0) {
            cout << "0\n"; // 先手必败，无法取胜
        } else {
            int cnt = 0;
            for (int i = 0; i < n; ++i) {
                // 若 res ^ a[i] < a[i]，则可以从第 i 堆取走 a[i] - (res ^ a[i]) 个
                if ((res ^ a[i]) < a[i])
                    cnt++;
            }
            cout << cnt << "\n";
        }
    }
    return 0;
}
```

## 4. 斐波那契博弈 (Fibonacci Nim)

**问题描述**  
有一堆石子 \(n\) 个，两人轮流取。先手第一次可以取任意多个（但不能取完）。之后每次取的石子数不能超过上一次对手取的 **2 倍**。取走最后一个者获胜。

**结论**  
若 \(n\) 是斐波那契数，则后手必胜；否则先手必胜。  
*策略：* 先手将 \(n\) 分解成若干不连续的斐波那契数（齐肯多夫定理），第一次取走最小的那个，之后模仿对手。

```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    set<int> fib;
    int f[50];
    f[0] = 1; f[1] = 1;
    fib.insert(1);
    for (int i = 2; i < 45; ++i) {
        f[i] = f[i-1] + f[i-2];
        fib.insert(f[i]);
    }
    int n;
    while (cin >> n && n) {
        if (fib.count(n))
            cout << "Second win\n";
        else
            cout << "First win\n";
    }
    return 0;
}
```

## 补充：SG 函数

对于更一般的博弈问题，通常使用 **SG 函数** 求解。每个局面的 SG 值定义为所有可达局面的 SG 值集合的 mex（最小非负整数）。多个独立子游戏组合的 SG 值为各子游戏 SG 值的异或。  
当且仅当 SG 值为 0 时，当前局面为必败态。
