## 扩展欧几里得算法（通用方法）

```cpp
// 扩展欧几里得算法
long long exgcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }
    long long d = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return d;
}

// 求逆元
long long inv(long long a, long long p) {
    long long x, y;
    long long d = exgcd(a, p, x, y);
    if (d != 1) return -1; // 逆元不存在
    return (x % p + p) % p; // 保证结果为正数
}
```

# 奇因子个数

## 数学原理详解

### 1. 数论基本定理

任何正整数都可以唯一分解为质因数的乘积：

text

n! = 2^k × 3^a × 5^b × 7^c × ... × p^z

其中 k, a, b, c, ..., z 是各质因数的指数。

### 2. 因子个数的公式

对于一个数 N = p₁^e₁ × p₂^e₂ × ... × pₙ^eₙ，它的因子总数为：

text

总因子数 = (e₁ + 1) × (e₂ + 1) × ... × (eₙ + 1)

应用到 n! 的情况：

text

n! 的总因子数 = (k + 1) × (a + 1) × (b + 1) × ... × (z + 1)

### 3. 奇因子的定义

一个因子是奇数的**充分必要条件**是它**不包含质因子2**。

换句话说，奇因子就是从 n! 的分解式中**去掉质因子2**后剩下的部分的所有因子。

### 4. 奇因子的个数

从 n! 的分解式中去掉质因子2后，我们得到：
m = 3^a × 5^b × 7^c × ... × p^z  （m 是奇数）

那么 m 的因子个数就是：
奇因子个数 = (a + 1) × (b + 1) × (c + 1) × ... × (z + 1)

### 5. 概率计算


奇因子的概率 = 奇因子个数 / 总因子个数
概率 = [(a + 1) × (b + 1) × ... × (z + 1)] / [(k + 1) × (a + 1) × (b + 1) × ... × (z + 1)]
     = 1 / (k + 1)
### 6. 关键观察

**分子分母中的 (a+1), (b+1), (c+1), ... 全部约掉了！**
最终结果只依赖于 **n! 中质因子2的指数 k**。
**计算 k 的公式**：`k = n/2 + n/4 + n/8 + n/16 + ...`
这个公式的直观理解：
- 在 1 到 n 中：
    - 有 n/2 个数能被 2 整除（贡献至少1个2）
    - 有 n/4 个数能被 4 整除（贡献额外的1个2）
    - 有 n/8 个数能被 8 整除（贡献额外的1个2）
    - 以此类推...
    -
注意：*fac[n] 是模 p 后的结果，不是真实的 n! 值，所以不能通过 a = fac[n] 来计算2的指数
即使 fac[n] 是真实值，当 n 很大时，直接对 a 进行位操作也不可行

## 数位dp(进位)

 前提准备：
 1. pp[0~9]：数字0-9的概率/计数（七段管题是概率，纯计数题全为1）
 2. num[0~3]：C的四位拆分 [个位, 十位, 百位, 千位]
 3. mod：模数（如998244353，计数题可省略）
 4. 
```cpp
constexpr i64 mod = 998244353; 
vector<vector<i64>> dp(5, vector<i64>(2, 0));
dp[0][0] = 1; 
// 初始状态：未处理，无进位，概率/计数=1
//dp[i][j]：处理完前i位（0=未开始，1=个位，2=十位，3=百位，4=千位），进位j(0/1)的概率/计数
// 逐位处理：从个位→十位→百位→千位（共4位）

for (int i = 0; i < 4; i++) {
    vector<vector<i64>> ndp(5, vector<i64>(2, 0)); // 新状态，每次重置
    for (int j = 0; j < 2; j++) { // 枚举当前进位：0/1
        if (dp[i][j] == 0) continue; // 无贡献，跳过
        // 枚举当前位A、B的数字（0-9）
        for (int a = 0; a < 10; a++) {
            for (int b = 0; b < 10; b++) {
                int sum = a + b + j;    // 当前位和 = A位 + B位 + 上一位进位
                int cur_digit = sum % 10; // 当前位结果（要匹配C的对应位）
                int new_carry = sum / 10; // 新进位（0/1，两数相加最大进位1）
                if (cur_digit == num[i]) { // 匹配C的第i位，才转移
                    ndp[i+1][new_carry] = (ndp[i+1][new_carry] + dp[i][j] * pp[a] % mod * pp[b] % mod) % mod;
                    // 纯计数题去掉模运算和pp：ndp[i+1][new_carry] += dp[i][j];
                }
            }
        }
    }
    swap(dp,ndp); // 更新状态为新状态
}

i64 ans = dp[4][0]; // 答案：处理完4位，最终无进位的概率/计数
```

## 最大异或段与问题

给定一个长度为 `n` 的非负整数数组 `arr`。  
你可以将数组切分成**任意多段**（至少1段），然后：
1. 对每一段内的所有数字求**异或和**（`xor`）；
2. 将所有段的异或值进行**按位与**（`&`）。
求所有可能切分方案中，按位与结果的最大值。

```cpp
void solve() {  
    int n;  
    cin >> n;  
    vector<i64> a(n + 1);  
    for (int i = 1; i <= n; i++) cin >> a[i];  
  
    for (int i = 2; i <= n; i++) a[i] ^= a[i - 1]; //前缀异或  
  
    i64 ans = a[n];  
    for (int i = 0; i <= n; i++)  
        ans = max(ans, a[i] & (a[n] ^ a[i])); //前缀异或后缀  
    cout << ans << '\n';  
}
```
  **最大答案只可能是以下两种之一：**

- **整个数组的异或和**（不切，或等价于奇数段合并）；
- **某个切分点的前缀异或 & 后缀异或**（只切一刀，两段）。

 ```cpp
vector<vector<i64>> mul(const vector<vector<i64>> &A, const vector<vector<i64>> &B) {  
    // 初始化结果矩阵 C 为 3x3 全零矩阵  
    vector<vector<i64>> C(3, vector<i64>(3, 0));  
    // 三重循环执行标准矩阵乘法：C[i][j] = sum_{k=0}^{2} A[i][k] * B[k][j]  
    for (int i = 0; i < 3; i++) {  
        for (int j = 0; j < 3; j++) {  
            for (int k = 0; k < 3; k++) {  
                // 累加并立即取模，防止中间结果溢出  
                C[i][j] = (C[i][j] + A[i][k] * B[k][j]) % mod;  
            }  
        }  
    }  
    return C;  
}  
  
// 矩阵快速幂：计算矩阵 m 的 e 次幂（对 mod 取模）  
vector<vector<i64>> mpow(vector<vector<i64>> m, i64 e) {  
    // 初始化结果矩阵 res 为单位矩阵 I（对角线上为 1，其余为 0）  
    // 因为任何矩阵乘以单位矩阵都等于自身  
    vector<vector<i64>> res(3, vector<i64>(3, 0));  
    for (int i = 0; i < 3; i++) res[i][i] = 1;  
  
    // 快速幂算法，将指数 e 按二进制位处理  
    while (e) {  
        // 如果当前最低位为 1，则将当前矩阵 m 乘到结果上  
        if (e & 1) res = mul(res, m);  
        // 将 m 自乘（平方），准备处理下一位  
        m = mul(m, m);  
        // 右移一位，相当于除以 2        e >>= 1;  
    }  
    return res;  
}
 ```
